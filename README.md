## About

**ditup** is an open-source initiative building a decentralized social network where you:

- Find like-minded folks
- Share ideas and issues
- Build projects together
- Own your data

This is the main monorepo containing the core [applications](./apps/), [packages (TODO)](.), and [documentation](./docs/).

## Technology Stack

Building on top of Universal Social Network - a decentralized network independent of companies, platforms and apps, where members own their identity and data.

- [Solid Protocol](https://solidproject.org/) is used for decentralized online identity and data storage.
- [ActivityPub](https://activitypub.rocks/) is used for communication between decentralized actors.

Read more at [solid-contrib/activitypub-interop](https://github.com/solid-contrib/activitypub-interop/).

## Getting Started

You need node v20 or newer.

This is an [npm workspaces](https://docs.npmjs.com/cli/v11/using-npm/workspaces) monorepo.

- clone this repository
- install dependencies with `npm ci`
- run the app in dev mode `npm run dev -w apps/ditup/`
- run the docs in dev mode `npm run dev -w docs/`

Or go to https://ditup.org and https://docs.ditup.org, and try it out.
