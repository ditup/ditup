# Localization

See [webawesome localization](https://webawesome.com/docs/localization) for inspiration.

Localization has two parts:

- user interface - labels, etc. - this can be achieved by available i18n methods and libraries
- linked data - with language tags, when language tags are available

The localization may react to <html lang="value"> using MutationObserver and switch the translations when the lang attribute changes.
